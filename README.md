# Beacon
Beacon is an free, open source 64-bit operating system
## Build
First, download the source code. Then run the following in the command line on the **root folder** in a Unix shell: <br/>
<code>make build</code><br/>
if there is any missing packages it will show up and just install it using:<br/>
<code>sudo apt-get install &lt;package-name&gt;</code><br/><br/>
You can use [Rufus](https://rufus.ie/) if you want to actually intall it on your computer <br/>
Or use tools like [Virtual Box](https://www.virtualbox.org/) or [Qemu](https://www.qemu.org/) to emulate the operating system<br/>
The ISO file is located in the <code>out</code> folder
